在这个 Java 项目里实现一个表达式求值器，并复用现有的 Calc 类：1. 先给 Calc 类补上缺失的 subtract、multiply、divide 方法，其中 divide 除以零时抛 ArithmeticException。2. 新建 Expr 类，提供 public static double eval(String expr)，要支持 + - * / 四则运算、括号、以及运算符优先级；四则运算的实际计算要调用 Calc 的对应方法；空表达式或含非法字符时抛 IllegalArgumentException。3. 新建 Main 类，在 main 方法里打印至少 5 个测试用例的结果，例如 eval("2 + 3 * (4 - 1)") 应为 11.0、eval("(1 + 2) * 3") 应为 9.0。4. 最后用 mvn 确保整个项目编译通过。
